This matrix is the release-truth surface for `v0.1-alpha`. A lane should only
be claimed as supported if it has current evidence here.

## Status Key

- `PASS` — validated in the current release window
- `PARTIAL` — some proof exists, but not enough for a clean public support claim
- `EXPERIMENTAL` — implemented and maybe CI-covered, but not yet validated
- `OUT` — not in the `v0.1-alpha` support claim

## Required Checks Per Claimed Lane

- install from release-shaped artifact or equivalent bundle layout
- host discovery / scan
- plugin instantiate
- parameter visibility
- GUI open
- GUI close
- short transport or audio proof
- known-issues reconciliation

## Current Release Read

If `v0.1-alpha` published today, the strongest honest claim is:

- supported: macOS + REAPER + VST2
- experimental: Windows, Linux, VST3, AU v2, 32-bit

Broader claims need fresh evidence first.
